== Overview

`obli-riscv-dev-kit` is a development kit for RISC-V architecture projects, built according to the Hyperpolymath Standard language policy.

=== Technology Stack (Planned)

[cols="1,2"]
|===
| Category | Technology

| Systems/Performance
| Rust, WASM

| Application Code
| ReScript

| Runtime
| Deno

| Package Management
| Guix (primary), Nix (fallback)

| Configuration
| Nickel

| State/Meta
| Guile Scheme
|===

=== Policy Enforcement

This repository enforces the Hyperpolymath Standard via CI:

[horizontal]
Allowed:: ReScript, Deno, Rust, WASM, OCaml, Haskell, Guile/Scheme, Ada, Gleam, Julia
Blocked:: TypeScript, Go, npm/yarn/bun, Python (except SaltStack)
